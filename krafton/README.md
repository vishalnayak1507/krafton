# Multiplayer Coin Collector

## Description
A real-time multiplayer game using Node.js and WebSockets.
- Server Authoritative architecture.
- 200ms simulated latency.
- Client-side entity interpolation for smoothness.

## Setup
1. Install Node.js.
2. Run `npm install` to install dependencies (ws, uuid).

## How to Run
1. Start the server:
   node server.js
2. Open `index.html` in two different browser tabs/windows.
3. Use Arrow Keys to move.
